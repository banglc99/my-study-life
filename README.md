# My Study Life 📚

**Ứng dụng quản lý lịch học và thời gian nghỉ ngơi thông minh dành cho học sinh**

## 🎯 Mục đích
Giúp học sinh:
- Lập lịch học hợp lý
- Quản lý thời gian nghỉ ngơi hiệu quả
- Nhận gợi ý bài tập và vận động
- Tránh sử dụng máy tính quá nhiều (đặc biệt trong kỳ hè)
- Cân bằng giữa học tập và sức khỏe

## 🛠️ Tech Stack

### Frontend - Web App
- **Framework**: React + TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit / Zustand
- **Routing**: React Router v6

### Frontend - Mobile App
- **Framework**: React Native + TypeScript
- **Navigation**: React Navigation
- **State Management**: Redux Toolkit / Zustand

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB / PostgreSQL
- **Authentication**: JWT
- **Real-time**: Socket.io (cho notifications)

## 🚀 Bắt đầu

### Yêu cầu
- Node.js >= 16
- npm hoặc yarn
- Git

### Cài đặt

```bash
# Clone repository
git clone https://github.com/banglc99/my-study-life.git
cd my-study-life

# Cài đặt dependencies
npm install

# Chạy development
npm run dev
```
